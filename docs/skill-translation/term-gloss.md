# 行銷術語中文註解速查表

skill 中出現的專有術語、方法論、心理學效應，**首次出現**時應於括號內加中文註解。
此表供批次改 skill 時參考，確保註解用詞一致。

**使用規則：**
- 術語原文保留（英文在前、中文註解在後的括號內）
- 首次出現在該檔必加註解；同檔後續出現可省略
- 中文註解以台灣繁中為準
- 若有多種常見譯法，以本表為主、避免同檔漂移

## 使用方式

1. 改 skill 時，**掃過每個檔案**，找到術語首次出現的位置
2. 依本表格式加註解
3. 同檔後續出現不重複加
4. 若遇本表未收錄的術語，先查 terminology.md；若仍無，不自行翻譯，**記到本表底部待補**
