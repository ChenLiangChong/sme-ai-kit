# Skill 翻譯術語對照表（台灣繁中）

本文件定義 `.claude/skills/` 底下所有模組在翻譯時應使用的統一術語。

**使用規則：**
- 所有譯文以本表為準，不得自行變更用詞
- 括號內為保留原文的時機（例如首次出現、圖表欄位、縮寫）
- 台灣繁中為準，遇到中國簡中用語必須改寫

---

## 一、翻譯原則

1. **台灣用詞優先**：「軟體 / 影片 / 專案 / 優化 / 網路 / 部落格 / 伺服器 / 介面 / 資料 / 滑鼠」
2. **避免中國簡中**：「软件 / 视频 / 项目 / 最佳化（可用）/ 网络 / 博客 / 服务器 / 界面（可用）/ 数据 / 鼠标」
3. **專有名詞保留英文**：品牌、產品、框架名稱（Salesforce、Gong、April Dunford）
4. **縮寫保留英文**：TAM / SAM / SOM / ICP / MQL / ROI / CTA / KPI / SEO / SaaS / B2B 等
5. **不自創譯名**：若對照表沒有的術語，沿用原文並以括號加譯，等待擴充本表
